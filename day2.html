Recap

Today is our first encounter with Ruby. We start the day off with a little demo from Wouter: - guess the number. With every try the program would tell you to go higher or lower. From this simple program we learned to use if else statements. Throughout the day we would learn more about how this game worked and actually program the game ourselves.

After the lunchbreak we had another assignment: make a simple store. The store should contain several items with different properties e.g. name, price and description. A menu had to be generated where the user could select the different items.
