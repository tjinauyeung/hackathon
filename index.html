Recap

Today we start off with object oriented programming. This ment that we would have to use classes to identify different objects in Ruby. By using this we could seperate a complex program into different smaller sized structures with their own behaviour. It's actually mimicking the real world. The example that we made was that of a bar.

So a bar consists of three different classes: Class Bartender, Class Customer and Class BeerEach class has their own methods and behaviour. By practising on naming classes and methods we practise in clarifying.
